Detailed Description

File where everything related to the pdf generation is defined.

In this file, we define PDFGenerator, the class that allows us to generate a PDF Report of the simulation. We also define PDFGeneratorVar, a struct class that regroup all variable needed to generate the PDF Report.

Author
: Romain Durieux and Nathan Trouillet (2022)
