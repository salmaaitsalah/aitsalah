\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_linker_1_1_linker}{Linker}} Class. 

This class is used to retrieve agent context data. This involves saving a history of context data and a history of QoS values. This class also defines to the agent when no more data is given, the different QoS calculations and the different evaluations.
